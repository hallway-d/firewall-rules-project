# Rules App

## Overview
A Django-based application for generating Suricata-compatible firewall rules.

## Features
- Accepts multiple sources, destinations, and ports.
- Generates unique SIDs for rules.
- Supports `pass` and `reject` actions.

## Requirements
- Python 3.11
- Django 5.x
